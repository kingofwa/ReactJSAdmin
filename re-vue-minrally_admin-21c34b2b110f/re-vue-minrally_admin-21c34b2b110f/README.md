# MINRALLY Admin

1. Install packages

```
yarn install
```

2. Start dev server

```
yarn start
```

3. Build project

```
yarn build
```

4. Deployment
  
- test

```
Build the project and push it to release/test branch
```

- staging

```
Build the project and push it to release/staging branch
```

- production

```
Build the project and push it to release/production branch
```
